## Setup

### Install from source

1. Install the SF CLI.

2. Clone the repository.

3. Install npm modules: `yarn install`

4. Link the plugin: `sfdx plugins:link .`

## Usage

<!-- commands -->
* `sf psrc compile --help`
* `sf psrc decompile --help`
